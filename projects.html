Core Systems Infrastructure

AnyStudio — Rust IDE for Android Development

A native desktop Integrated Development Environment engineered in Rust and the egui immediate mode GUI library. This system fundamentally shifts the Android build paradigm by compiling apps via a direct DEX pipeline.

  • Implemented a custom anystudio-dex crate capable of writing classes.dex binaries directly, entirely bypassing the Java compilation step and Gradle daemon.
  • Architected separate modular crates for the block compiler, APK packaging pipeline, ADB device wrapper, and Android SDK tool management.

Technologies: Rust egui Dalvik Executable Format

Webelo — Browser Rendering Engine
Repository

A ground-up implementation of a Document Object Model library and HTML visualizer written in C++. The engine is engineered to strictly follow the WhatWG DOM Standard.

  • Implemented the full DOM tree structure with rigorous preorder depth-first traversal algorithms for rendering and event propagation.
  • Serves as the foundational architecture for a complete custom browser rendering engine.

Technologies: C++ WhatWG Standards
