Android DEX Pipeline & Rust

Notes on AnyStudio

As part of AnyStudio (a native desktop Android IDE built in Rust and egui), I needed a way to compile code without relying on Java or Gradle.

I built the custom `anystudio-dex` crate in Rust. This crate allows the IDE to write `classes.dex` files directly, bypassing Java compilation entirely.

The architecture separates the block compiler, the APK build pipeline, the ADB wrapper, and Android SDK tool management into distinct Rust crates.
